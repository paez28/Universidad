template<class G>
class ed::OperadorNodo< G >

Interfaz para definir operaciones sobre un nodo de un Arbol.

La clases derivadas de OperadorNodo definirán operaciones para el tratamiento de los nodos de un Arbol cuando se realiza un recorrido sobre el mismo.

  • G define el campo de información de los nodos del Arbol.
